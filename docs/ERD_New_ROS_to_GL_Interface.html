7. Rumus Kalkulasi

Cost Book Material (di New ROS)

total_value_rp = total_value_usd × kurs_pib
bm_rp = total_value_rp × bm_persen / 100
forwarding_cost = 200 × kg_unit
total_nilai_inventory = total_value_rp + bm_rp + prorate_ls + forwarding_cost + prorate_insurance + prorate_others
cost_book = total_nilai_inventory / kg_unit

Moving Average (di Warehouse Stock)

saldo_lama = qty_stock_lama × harga_beli_lama
nilai_baru = qty_masuk × cost_book
qty_akhir = qty_stock_lama + qty_masuk
total_nilai = saldo_lama + nilai_baru
costbook_baru = total_nilai / qty_akhir

Price Per Coil

price_per_coil = berat_bersih_coil × cost_book_material

Jurnal GL

Jurnal A (Close ROS):
DEBET: Persediaan In-Transit (1105-01-03) = total_nilai_inventory semua material
KREDIT: DP + BM + LS + Forwarding + Insurance + Others + Selisih Kurs + Pembulatan

Jurnal B (Finalize Incoming):
DEBET: Σ price_per_coil per gudang (COA 1105-01-01 Produksi atau 1105-01-02 Slitting)
KREDIT: Σ price_per_coil total (COA 1105-01-03 In-Transit) — membalik Jurnal A

Total DEBET = Total KREDIT (balanced per jurnal)
